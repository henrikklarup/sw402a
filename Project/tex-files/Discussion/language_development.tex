\section{Compiler language}
The MASSIVE compiler is developed in the C\# language, this is done because C\# is an object oriented language and therefore provides a good base to create classes and objects used in the compiler. E.g. the abstract syntax trees.\\
Java provides the same base as an object oriented language, but due to developing the GUI in C\# in Visual Studio, it is convenient to create the MASSIVE compiler in the same programming language.
